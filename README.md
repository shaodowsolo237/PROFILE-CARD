PROFILE CARD PROJECT

Aclean and modern *Profile Card* built using *HTML* and *CSS* .

This project showcases basic front-end skills including layout design, flexbox/grid usage, responsive styling, and clean UI presentation.

FEATURES

-Responsive layout
-Profile image with circular design
-User name and bio section
-Social media links with hover effects
-Minimalist and clean visual style

TECHNOLOGY USED

-HTML5
CSS3(Flexbox)
